# 如何保存和恢复对话历史

## 问题说明

VS Code的AI助手对话默认不会自动保存到项目中，每次重新打开可能需要重新开始对话。

---

## 解决方案

### 方案一：使用对话导出功能（推荐）

**在Cline/Clime中**：

1. 点击对话窗口右上角的菜单（三个点）
2. 选择 "Export Chat" 或 "导出对话"
3. 保存为 `.md` 文件到项目目录

**保存位置建议**：
```
d:/test111/docs/chat-history/
├── 2024-03-29-iot项目讨论.md
├── 2024-03-29-网络配置.md
└── 2024-03-29-测试指南.md
```

---

### 方案二：创建项目文档总结

在项目根目录创建一个总结文档，记录关键信息：

<write_to_file>
<path>d:/test111/项目对话总结.md</path>
<content># IoT智能农业项目 - 对话总结

## 关键配置信息

### 服务器地址
- **HarmonyOS模拟器**：`10.0.2.2:3000`
- **ESP32设备**：本机IP（如 `192.168.1.105:3000`）

### 启动命令
```bash
cd d:/test111/net/HarmoAgri/tools
node esp-mock-server.js
```

### ESP32配置位置
- 文件：`iottest/src/config.h`
- 需要配置：WiFi名称、WiFi密码、服务器IP
