# ESP32 硬件连接表

## 引脚调整

- `HC-SR501` 已从 `GPIO12` 调整到 `GPIO14`
- 原因：`GPIO12` 是 ESP32 启动相关脚位，外设上电状态容易影响启动稳定性

## 供电连接建议

| 部分 | 建议供电 | 说明 |
| --- | --- | --- |
| ESP32 开发板 | 稳定 `5V` 接 `VIN/5V`，或稳定 USB 供电 | 不要把外部 `5V` 接到 `3V3` |
| 3.3V 传感器 | 由 ESP32 `3V3` 输出供电 | 适用于 DHT11、SGP30、HW-390、雨滴、光敏、DS18B20 |
| 继电器模块 | 独立 `5V` 电源 | 不建议直接吃 ESP32 板载 `5V` |
| 水泵 | 按泵额定电压独立供电 | 常见为 `5V`/`12V`，不要从 ESP32 取电 |
| 补光负载 | 按负载额定电压独立供电 | 继电器只负责开关，不负责给负载降压 |

## 必须满足的接线规则

- `ESP32 GND` 必须与继电器电源 `GND`、水泵电源 `GND` 共地
- 水泵和补光负载不要直接从 ESP32 供电
- 继电器模块优先用独立 `5V`
- 水泵属于感性负载，建议在泵两端加续流二极管
- 水泵电源线和传感器信号线不要并排走线
- 雨滴传感器、土壤湿度传感器的信号线尽量短

## 推荐电源拓扑

| 电源 | 连接对象 |
| --- | --- |
| `5V-A` | ESP32 `VIN/5V` |
| `3.3V` | 由 ESP32 稳压后提供给低功耗传感器 |
| `5V-B` | 继电器模块 |
| `Pump Power` | 水泵本体，按额定电压单独供电 |
| `Light Power` | 补光灯本体，按额定电压单独供电 |
